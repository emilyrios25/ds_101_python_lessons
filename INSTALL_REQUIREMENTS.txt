Installation Requirements
========================================

SYSTEM REQUIREMENTS
-------------------
- Python 3.8 or higher (recommended: Python 3.9+)
- pip package manager
- Internet connection for downloading packages and models
- 2GB+ free disk space for all packages and models

COMPATIBLE OPERATING SYSTEMS
-----------------------------
- Windows 10/11
- macOS 10.15+ (Catalina or newer)
- Linux (Ubuntu 18.04+ or equivalent)

---------------

- matplotlib      # Basic plotting and visualization

Interactive Visualization:
- plotly          # Interactive charts and maps

Web Scraping (Lesson 1):
- praw            # Python Reddit API Wrapper

- scipy           # Scientific computing

Utilities:
- tqdm            # Progress bars

------------------
- punkt           # Sentence tokenization (Lesson 4)
- vader_lexicon   # Sentiment analysis lexicon (Lesson 5)

SPACY LANGUAGE MODELS
---------------------
- en_core_web_md  # Medium English model (earlier lessons)
- en_core_web_trf # Transformer English model (advanced lessons)

EXTERNAL DATABASES
------------------
- GeoNames Database # Geographic name database (auto-downloaded by geoparser)

INSTALLATION METHODS
--------------------

Method 1 - Automated Installation (Recommended):
Run the provided installation script:
    python installs_required.py

Method 2 - Manual Installation:
Install packages individually using pip:
    pip install pandas numpy matplotlib plotly mapclassify
    pip install praw requests beautifulsoup4 tqdm jupyter ipywidgets
    pip install nltk spacy geoparser transformers torch scipy
    
Download NLTK data:
    python -c "import nltk; nltk.download('punkt'); nltk.download('vader_lexicon')"
    
Download spaCy models:
    python -m spacy download en_core_web_md
    python -m spacy download en_core_web_trf
    
Setup geoparser database:
    python -m geoparser download geonames

TROUBLESHOOTING
---------------

Common Issues:
1. Python version too old - Upgrade to Python 3.8+
2. pip not found - Install/update pip
3. Network timeouts - Retry installation, check internet connection
4. Disk space - Ensure 2GB+ free space for all models
5. Permission errors - Run as administrator (Windows) or use sudo (macOS/Linux)

For spaCy model download issues:
    python -m spacy validate
    
For geoparser database issues:
    python -c "from geoparser import Geoparser; Geoparser()"
